# OpenCampus Hub

An open-source platform where students can explore academic resources across domains and stay updated with college events in one place.

## Problem

Students rely on multiple platforms to find study resources and college event updates, leading to fragmented information and unnecessary confusion.

## Solution

OpenCampus Hub centralizes curated academic resources and college-specific event updates into a single, accessible platform designed for students.

## Features

- Resource hubs for different academic domains
- Centralized college-wise event listings
- Simple, accessible, non-tracking design
- Open-source and community-driven

## Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- Data: JSON (temporary)
- Version Control: Git & GitHub
